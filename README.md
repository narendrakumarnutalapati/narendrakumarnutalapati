### LICITRA Research Program

Cryptographic runtime integrity for agentic AI systems.

**LICITRA-SENTRY** — Five-gate authorization pipeline with Ed25519 execution tickets, mandatory tool mediation, and CT-style witnessed transparency. 13 reproducible experiments. Maps to 9/10 OWASP Agentic Top 10 categories.

**LICITRA-MMR** — Merkle Mountain Range audit ledger with logarithmic verification (~14 ops for 10K events).
